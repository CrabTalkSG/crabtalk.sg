CRAB TALK WEBSITE UPLOAD STEPS

This ZIP uses your existing image files already in GitHub under:
public/images

Do NOT delete public/images.
Do NOT re-upload the image folder if the image files are still there.

Upload only these root files from this ZIP:
1. index.html
2. package.json
3. vercel.json
4. robots.txt
5. sitemap.xml

GitHub steps:
1. Open https://github.com/CrabTalkSG/crabtalk.sg
2. Make sure you are on the main branch.
3. Click Add file > Upload files.
4. Drag the 5 files above into the upload box.
5. At the bottom, click Commit changes.
6. Wait for Vercel to deploy.
7. In Vercel, the newest deployment should show Ready.
8. Open https://www.crabtalk.sg/ in an incognito/private window to check.

Important:
- This package.json copies your existing public/images folder into the Vercel build output.
- The Google review text is static and typed from your screenshots.
- The buttons link customers to Google Maps and the official Google review form for the latest live reviews.
